Programming Languages

A programming language is a formal language which comprises a set of instructions used to produce various kinds of output. Programming languages are used in computer Programming to create programs that implement specific algorithims.




Java

Java is a general-purpose computer-programming language that is consurrent, class-based object-oriented and specificaly designed to have as few implication dependencies as possible. In Java programms are not compiled into executable files; they are compiled into bytecode (as discussed earlier) which the jvm (Java Virtual Machine) then executes at runtime.

Python

Python is an interpreted, high-level, general-purpose programming language. Created by Guido Van Rossum and first realesed in 1991, Python has a design philosiphy that emphasizes codee readability, notability using significant whitespace. It provides, constructs that enable clear programming on both small and large scales.
